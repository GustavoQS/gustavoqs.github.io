Este é meu PRIMEIRO documento de HTML.

Esse é tipo o título maior ("h1"), vai indo h1, h2 , h3 , h4... como se fosse tamanhos

Nesse meu tutorial, as aspas vão significar < ou > , (as tags).

Parágrafos e Títulos (usei tag "h2" aqui)

Esse é a tag com "p", significa parágrafo, bem auto explicativo.

Os títulos usam a tag "h1" (maior letra), "h2", "h3", "h4"...

Se eu quiser parágrafos com algumas linhas pra baixo (igual o espaçamento entre linhas acimas), é só criar outra tag "p" embaixo da anterior.

Para fazer uma parágrafo abaixo de outro sem pular linha,
é so colocar a tag "br /", pode colocar no meio dos "p".

Para criar uma linha igual a acima, é so colocar a tag "hr /".

Palavras Que São Links

Basta usar a tag (sem o espaço após o < ) < a href="http://sitedesejado.com">Palavra Desejada! (sem o espaço antes do "a"). Exemplo: CLIQUE AQUI PARA ABRIR O GOOGLE!

Tópicos ou Listas

Para criar tópico de bolinha, cria a tag "ul", e na linha debaixo, cria a tag "li" e feche as duas.
Para criar tópico de número, no lugar da tag "ul", coloca a tag "ol" e repita os outros processos. EXEMPLO:

  • Tópico com bolinha (tag "ul")
  • Dentro da "ul" tem a "li"
  1. Tópico com Número (tag "ol")
  2. Dentro da "ol" tem a "li"

Agrupar textos

Para agrupar os textos, basta criar a tag "div". Dentro dela, você pode colocar textos, e até mesmo outras tags.
NÃO pode iniciar uma tag dentro da "div" e fechar depois do final da "div"

Letras com cores

Para colocar letras com cor basta colocar antes da palavra que você deseja colorir a tag, (sem os espaços das tags): < span class="cor desejada">Palavra desejada< /span>
Se não funcionar, não se preocupe, não consegui fazer funcionar ainda também.

Imagens

Para colocar imagens, basta usar a tag (sem o espaço após a primeira < ) < img src="c:caminho/do/arquivo/nomedaimagen.formatoDaimagen" width="PixelsDESEJADOSpx" height="PixelsDESEJADOSpx" />
